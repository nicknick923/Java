D. Add a handler for keyPressed for the figPanel
      - while in design mode, select FigPanel 
      - Right click on figPanel, choose Properties and then Events
      - To the right of KeyPressed, select the "..."
      - Click the "Add..." button, name the handler, myKeyAdapter
      - In Source, window Copy/Paste the following to the myKeyAdapter handler:

      stickfig.hide();
      if ( evt.getKeyCode() == java.awt.event.KeyEvent.VK_DOWN )
         stickfig.move(0, 10);
      else if ( evt.getKeyCode() == java.awt.event.KeyEvent.VK_UP )
         stickfig.move(0, -10);
      else if ( evt.getKeyCode() == java.awt.event.KeyEvent.VK_LEFT )
         stickfig.move(-10, 0);
      else if ( evt.getKeyCode() == java.awt.event.KeyEvent.VK_RIGHT )
         stickfig.move(10, 0);
      stickfig.draw();
      picfig.draw();
